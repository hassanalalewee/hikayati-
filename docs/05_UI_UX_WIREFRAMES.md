# Hikayati — UI/UX Wireframes & Design System

**Version:** 1.0 | **Date:** 2026-06-13

---

## 1. DESIGN PRINCIPLES

1. **Arabic-First:** Every element designed RTL from the ground up, not adapted
2. **Warm & Trustworthy:** Colors and typography signal safety and care
3. **Parent-Centric Interface, Child-Centric Output:** The parent uses a clean SaaS UI; the story output feels like a premium children's book
4. **Progressive Disclosure:** Complex personalization options revealed gradually
5. **Celebration:** Every story completion is a micro-celebration moment
6. **Mobile-First:** 70%+ of users will be on mobile (GCC market research)

---

## 2. DESIGN SYSTEM

### 2.1 Color Palette

```
Primary:
  Indigo 600:    #4F46E5  — Primary actions, CTA buttons
  Indigo 50:     #EEF2FF  — Light backgrounds, selected states

Accent:
  Amber 400:     #FBBF24  — Stars, ratings, achievements, warmth
  Amber 50:      #FFFBEB  — Warm backgrounds

Success:
  Emerald 500:   #10B981  — Story complete, success states
  Emerald 50:    #ECFDF5  — Success backgrounds

Story Purple:
  Purple 600:    #9333EA  — Story cards, creative elements
  Purple 50:     #FAF5FF  — Story card backgrounds

Neutral:
  Slate 900:     #0F172A  — Primary text (Arabic)
  Slate 700:     #334155  — Secondary text
  Slate 400:     #94A3B8  — Placeholder text
  Slate 100:     #F1F5F9  — Background gray
  White:         #FFFFFF  — Cards, surfaces

Brand Gradient:
  from-indigo-600 via-purple-600 to-amber-500
  (Used in hero sections, story covers overlay)
```

### 2.2 Typography (Arabic-First)

```css
/* Primary Arabic Font */
font-family: 'Noto Sans Arabic', 'Cairo', sans-serif;

/* Headings */
--font-h1: 2.25rem / 700 weight / -0.02em letter-spacing
--font-h2: 1.875rem / 700 weight
--font-h3: 1.5rem / 600 weight
--font-h4: 1.25rem / 600 weight

/* Body */
--font-body-lg: 1.125rem / 400 weight / 1.8 line-height (Arabic needs tall lines)
--font-body:    1rem    / 400 weight / 1.8 line-height
--font-body-sm: 0.875rem / 400 weight

/* Story Text (children's book feel) */
--font-story:   1.25rem / 500 weight / 2.0 line-height (larger, more readable)
```

### 2.3 Spacing & Grid

```
RTL grid: 4-column (mobile), 8-column (tablet), 12-column (desktop)
Base unit: 4px
Container max-width: 1200px
Gutter: 24px (mobile), 32px (desktop)
Border radius: sm=4px, md=8px, lg=16px, xl=24px, full=9999px
```

### 2.4 Component Library (Shadcn + Custom)

Custom components:
- `<StoryCard>` — Animated story preview card with cover image
- `<ChildAvatar>` — Child profile display with growth ring
- `<GoalBadge>` — Goal selection pill with icon
- `<StoryProgress>` — Real-time generation progress stepper
- `<AudioPlayer>` — Custom Arabic-labeled audio player
- `<PDFViewer>` — In-browser story book viewer (page-flip)
- `<DevelopmentRadar>` — Radar chart of development dimensions
- `<MilestoneCelebration>` — Full-screen confetti achievement modal
